# react-textutils
A simple and efficient web app for text manipulation. Convert text to uppercase/lowercase, remove extra spaces, count words/characters, reverse text, and preview changes in real-time. Built with a clean, responsive UI.

---

## 🚀 Live Demo

https://react-textutils-seven.vercel.app/

---

## ✨ Features

- Convert text to uppercase or lowercase
- Capitalize sentences or words
- Remove extra spaces and formatting
- Count words and characters in real-time
- Reverse text
- Copy text to clipboard instantly
- Live preview of changes

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/Bhaktishree-Pakka/react-textutils.git
```

2. Navigate to the project folder:

```
cd react-textutils
```

3. Install dependencies:

```
npm install
```

4. Run the development server:

```
npm run dev
```
